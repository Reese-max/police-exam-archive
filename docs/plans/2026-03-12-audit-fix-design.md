## 修復細節

### F1: 106-107年國文選擇題複製（270題）

**事實**: 106-107年國文（作文、公文與測驗）是全學系共用考卷。以下學系已正確解析（各 10 題選擇題 + 2 題申論）：
- 公共安全學系情報組
- 國境警察學系移民組
- 犯罪防治學系矯治組（106+107）
- 犯罪防治矯治組（106+107，四等對應）

**做法**:
1. 以情報組的選擇題為來源（source of truth）
2. 遍歷 106年、107年所有國文 JSON
3. 若只有 essay 類型且 notes 包含「2B鉛筆」→ 判定為缺失
4. 將 10 題選擇題插入 questions 陣列（essay 題之後）
5. 新增 section: "乙、測驗題" 標記

### F2: 112-113年矯治組國文（20題）

**做法**:
1. 從考選部下載 112年、113年犯罪防治學系矯治組國文 PDF
2. 用 pdf_to_questions.py 解析選擇題部分
3. 將解析出的 10 題選擇題插入現有 JSON

### F3: 108年外國文誤標（40題）

**現狀**: 公共安全學系情報組/108年/外國文（英文）有 43 題全標為 essay，其中 Q1-Q40 應為 choice。

**做法**:
1. 從考選部下載 108年外國文 PDF
2. 將 Q1-Q40 的 type 改為 "choice"
3. 從 PDF 解析 options (A)(B)(C)(D) 和 answer

### F4: 英文克漏字空題幹（528題）

**現狀**: 這些都是閱讀測驗/克漏字填空題，有 passage 但 stem 為空。

**做法**:
- 對有 passage 的空 stem 選擇題，設定 stem 為 passage 開頭的提示文字（如「請依下文回答第X題至第Y題」已在 passage 中）
- 由於這些是克漏字，題幹本身就嵌在 passage 中（如 "___53___"），stem 設為描述性文字即可

### F5: 水上警察 Q54 缺失（1題）

**做法**:
1. 從考選部下載水上警察學系/106年/中華民國憲法與水上警察專業英文 PDF
2. 提取 Q54 的 stem、options、answer
3. 插入 JSON 正確位置

### F6: Metadata 滲入申論題幹（8筆）

**現狀**: 情報組 106-113年國文 Q二 尾部混入「乙、測驗部分:(20分) 代號:1101...」

**做法**:
- 正則匹配 `乙、測驗部分` 開頭的文字，截斷之後的內容
- 同時檢查 `代號:\d{4}` 模式一併清除

## 修復順序

1. **F6** Metadata 清理（最簡單，純正則）
2. **F4** 空題幹修復（程式修復，不需下載）
3. **F1** 國文選擇題複製（從現有 JSON，不需下載）
4. **F2** 下載 112-113年 PDF → 解析
5. **F3** 下載 108年外國文 PDF → 修正 type + 補 options
6. **F5** 下載 106年水上警察英文 PDF → 補 Q54

## 驗證計畫

每步修復後執行驗證：
- 確認 questions 陣列完整性（題號連續、type 正確）
- 確認 choice 題有 options (4個) 和 answer
- 確認 essay 題 stem 無 metadata 殘留
- 最終重跑全面掃描確認 0 問題
